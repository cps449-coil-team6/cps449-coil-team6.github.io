Project Description:

Our Project is a content-based Movie Recommendation System built using Python, Pandas, Scikit-learn, and served with a lightweight FastAPI web framework. We start with the TMDB 5000 dataset, which includes two CSV files movies.csv and ratings.csv. After merging these datasets on the movie title, we extract and clean key fields such as overview, genres, cast, crew, keywords, budget, runtime, and popularity.
